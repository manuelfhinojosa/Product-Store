# Product Store

A full-stack responsive product management application that enables users to **add**, **edit**, and **delete products** with a clean and intuitive interface. Built with modern JavaScript for both frontend and backend, this project demonstrates practical skills in web development, REST APIs, and real-world state management.

---

## Why This Project Matters

This project showcases the ability to:

- Design and implement a **full-stack web application**
- Build a responsive frontend with dynamic UI updates
- Create and consume a backend REST API
- Handle product CRUD (Create, Read, Update, Delete) operations
- Apply modular and maintainable JavaScript across client and server code

It reflects practical experience building applications similar to real world e-commerce admin dashboards or inventory tools.

---

## Core Features

- Product creation with dynamic form handling
- Product listing with live updates
- Edit and update existing products
- Delete products with confirmation
- Responsive UI for different screen sizes
- Client-server communication via REST API

## Getting Started

### 1. Clone the repository
- git clone https://github.com/manuelfhinojosa/Product-Store.git
- cd Product-Store

### 2. Install dependencies
- npm install

### 3. Run the application
- npm start

Visit your browser at http://localhost:3000 (or configured port) to use the product store.
